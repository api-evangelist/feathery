# Feathery

Feathery is an enterprise form SDK and AI-driven data intake platform built for financial services. It provides a REST API for creating multi-step forms, managing field logic, collecting submissions, and integrating with payment, authentication, and financial services systems including insurance and wealth management workflows.

## APIs

- **Feathery REST API** - RESTful API with token-based authentication supporting form management, submissions, document intelligence, hidden fields, end users, and workspace reporting. Multi-region: US, Canada, Europe, Australia.

## SDKs

- **feathery-react** - React client library (TypeScript): https://github.com/feathery-org/feathery-react

## Authentication

Token-based authentication via `Authorization: Token <API KEY>` header. API keys are environment-specific (production, development).

## Plans
